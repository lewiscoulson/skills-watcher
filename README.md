# React boilerplate 2016
A boilerplate to start a client-side project using React.

## Tech stack
- es2015
- react
- react-dom
- react-router
- webpack
- webpack-dev-server + hot module replacement
- babel 6
- sass

## Getting started
```bash
$ git clone git@github.com:rbartoli/react-boilerplate-2016.git
$ npm install
$ npm start
```
